# Ressources Guide

This folder contains the project documentation.

If you only read one file to understand how the project works, start with:

- `ressources/DATA_HANDLING.md`

It explains:
- how to run the app
- what the main data files are
- how they relate to each other
- what artifacts are
- how to rebuild and update runtime data
